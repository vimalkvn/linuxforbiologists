6.2.3. Creating a notebook¶

Open Zim by clicking on its icon in applications menu (Fig. 109). You can use the search bar to locate it quickly.

When you open the program for the first time, you will notice an Add Notebook dialog (Fig. 110).

Here, you can set a name for this notebook and select a directory to save notebooks.

You can accept the defaults.

Click on the OK button to proceed.

This will open the main window of the program (Fig. 111).
