5. Getting started with Galaxy¶

The Galaxy project enables researchers to perform computational analyses using a web interface.

You can use the service online at http://usegalaxy.org/ or any of the other available public servers.

The software powering the servers itself, is available under an open-source licence. You can download it and set up a local instance on your computer.
