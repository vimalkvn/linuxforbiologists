``head`` â€” print first few lines of a file
==========================================
With the ``head`` command, you can view the first few
(default: 10) lines of a file.

For example:

.. code-block:: bash

   head /usr/share/dict/words

Will produce the following output:

.. code-block:: console

   A
   A's
   AMD
   AMD's
   AOL
   AOL's
   AWS
   AWS's
   Aachen
   Aachen's
