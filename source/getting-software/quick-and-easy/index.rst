The quick and easy method
=========================
The *quick and easy* method is to use 
:term:`Software Manager`, to search and 
install software on your desktop. It supports installing 
software from Linux Mint, Debian and Flathub repositories.

Examples of software that can be installed using
this method:

`Circos <https://circos.ca>`_
   — plotter for visualizing data. 
   
`Clustal Omega <https://www.clustal.org/omega>`_
   — general purpose multiple sequence alignment program for
   nucleotide and protein sequences.
  
`EMBOSS <https://emboss.sourceforge.net>`_
   — European molecular biology open software suite.
  
`NCBI BLAST+ <https://blast.ncbi.nlm.nih.gov/Blast.cgi>`_
   — next generation suite of BLAST sequence search
   tools.
  
`Bioconductor <https://www.bioconductor.org>`_
   — tools for the analysis and comprehension of
   high-throughput genomic data.

`CD-HIT <http://weizhongli-lab.org/cd-hit/>`_
   — suite of programs designed to quickly group
   sequences.

`UGENE <http://ugene.net/>`_
   — integrated bioinformatics toolkit.
