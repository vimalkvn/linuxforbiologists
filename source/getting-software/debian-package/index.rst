Debian packages
===============
If your software of interest is available as a
:term:`Debian package`, you can use the :term:`gdebi` 
program to install it. If the software has any 
:term:`dependencies`, they will be installed 
automatically.

Examples of software that can be installed using this
method:

:term:`RStudio Desktop`
   — an IDE for the R programming language

:term:`Modeller`
   — a program for comparative modelling of protein 
   three-dimensional structures

:term:`MEGA`
   — Molecular Evolutionary Genetics Analysis
