Perl modules
============
For installing Perl modules, you can configure 
the :term:`local-lib` module to create and use a local 
directory. The default location is ``$HOME/perl5``.

If your software of interest is available on 
:term:`CPAN`, you can install it easily using 
the :term:`cpanminus` script.

.. note::

   Your software of interest, might be available using
   :doc:`../quick-and-easy/index`. 
   
   Search for package names containing ``perl``.
