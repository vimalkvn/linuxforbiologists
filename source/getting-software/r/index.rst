R packages
==========
If your software of interest is available as an R 
package on :term:`CRAN` or :term:`Bioconductor`, 
you can install it using the ``install.packages()`` 
or ``BiocManager::install()`` commands respectively.

.. note::
   
   Your software might already be available using
   :doc:`../quick-and-easy/index`. 
   
   Search for package names starting with 
   ``r-cran-`` or ``r-bioc-``.
