Running a Linux desktop
=======================
There are many ways [1]_ to run a Linux desktop on your
computer:

1. Install a Linux distribution
2. Run Linux as a virtual machine
3. Run Linux in the cloud

I will discuss option 2 in this book, as it is a lot easier
for new users to get started.
